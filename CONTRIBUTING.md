# Contributing

## Scope

This repository is for the toolkit only, not private sync data.

## Guidelines

- do not commit `config.json`
- do not commit `workspace/`
- do not commit plaintext secrets
- keep launcher behavior explicit and safe
- prefer user-facing wording that makes overwrite direction obvious

## Suggested Areas

- documentation improvements
- safer confirmation flows
- better path auto-detection
- stronger encryption implementation
- better status reporting
